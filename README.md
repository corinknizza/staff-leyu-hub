# staff-leyu-hub

本仓库用于归档和发布多个独立 HTML 页面。这些页面通常为静态内容，不依赖特定后端服务，可直接通过浏览器访问。

## 目录说明

```
├── index.html          # 主页（可选）
├── pages/              # 存放各独立 HTML 页面
│   ├── example1.html
│   └── example2.html
├── assets/             # 公共资源（CSS、JS、图片等）
├── README.md           # 本文件
```

- `pages/`：主要页面存放目录，每个 HTML 文件视为一个独立归档单元。
- `assets/`：页面共用资源，可按需组织子目录。

## 页面归档说明

- 所有页面均为独立 HTML，可直接在浏览器中打开运行。
- 页面间无强耦合关系，每个文件可独立维护和更新。
- 提交时请保持文件结构清晰，避免冗余依赖。

## 维护说明

- 欢迎提交新的独立页面或对现有页面进行改进。
- 提交前请确保页面无外部敏感链接或依赖。
- 请尽量保持代码简洁，避免引入未授权的第三方资源。
- 如有较大变更，建议在 commit 信息中简要说明改动内容。

## 其他

本仓库仅作为页面归档与展示用途，不提供在线服务或动态功能。如有问题或建议，欢迎通过 Issues 交流。
